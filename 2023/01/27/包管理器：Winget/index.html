介绍

Windows Package Manager（也叫 winget）是一个 Windows 软件包管理器，它由一个命令行工具和一组用于在 Windows 10 上安装应用的服务组成。winget 可以帮助使用者快速轻松地发现地安装不同的工具

安装

支持范围

WinGet 支持 Windows 10 1709 及以上版本

安装方式

Microsoft Store

  • 或是加入 Windows 10 Insider，并下载安装 Windows Insider 版本的系统
  • 或是通过官方注册通道来将自己加入 WinGet Preview flight ring

GitHub Release

通过WinGet 的官方 GitHub 仓库，在 Release 页面手动下载 WinGet 的安装程序进行手动安装

使用命令

在CMD或者PowerShell使用WinGet

WinGet CLI 设置

您可以通过编辑文件来配置 WinGet。运行将在默认 json 编辑器中打开该文件，如果没有配置编辑器，则使用记事.exe纸。settings.json``winget settings

文件位置

设置文件位于 %LOCALAPPDATA%=包\Microsoft.DesktopAppInstaller_8wekyb3d8bbwe状态\设置.json

如果您使用非打包的 WinGet 版本，通过从源代码构建它，该文件将位于 %LOCALAPPDATA% {微软}WinGet=Settings.json 下

源

这些设置涉及对 WinGet 源的配置。source

自动更新InterinMinutes

正整数表示更新间隔（以分钟为单位）。只有在使用源时，才检查更新。零将禁用对源更新的检查。任何其他值都无效。

  • 禁用： 0
  • 默认值： 5

手动更新源使用winget source update

视觉

这些设置涉及 WinGet 显示的可视元素visual

进度栏

WinGet 在参数未指定时显示的进度条的颜色。

  • accent (default) + 重音（默认）
  • retro + 复古
  • rainbow + 彩虹
